Gitの練習をしていきます。
基本的なワークフローを学びます。

・git add
・git commit
・git rm
・git status
・git diff

*************************************************
イニシャル
git init

ファイルの変更確認
git status

ステージング
git add .

ローカルリポジトリ(-vで差分を確認)
git commit -v

リポジトリの確認
git log

一行で表示する
git log --oneline

ファイルの差分を表示する
git log -p index.html

最新の指定数分のログを確認する(以下は3個)
git log -n 3


ファイルを削除した場合
git rm ファイル名


ファイルの変更差分
今のローカルの状態とステージングとの差分を確認
git diff

ステージングとコミットの差分
git diff HEAD
